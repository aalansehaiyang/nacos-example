## spring boot 集成nacos的服务发现
---

工程：




注册一个服务：

```
curl -X PUT 'http://172.16.113.76:8848/nacos/v1/ns/instance?serviceName=example&ip=172.16.113.76&port=8080'
```

查找一个服务：

http://localhost:8080/discovery/get?serviceName=example
